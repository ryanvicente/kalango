### Resumo

  (Apresentar um breve resumo sobre o seu trabalho, indicando o contexto e as principais funcionalidades.)
  O trabalho se trata de um sistema para republicas se apresentarem e automatizar estruturas internas.
  As principais funcionalidades são de automatizar o que existe via quadro ou outro esquema físico para estruturas das repúblicas.
  

<!-- Apresentar o tema. -->
### 1. Tema

  O trabalho final tem como tema o desenvolvimento de uma landpage com um sistema para automatizar e centralizar a estrutura de uma republica.

<!-- Descrever e limitar o escopo da aplicação. -->
### 2. Escopo

  Este projeto terá as seguintes funcionalidades:
  Uma landpage para apresentar a republica e afins.
  Um login para outros republicanos e outras pessoas.
  Uma página de administrador(um cargo mais alto da república).
  Uma página para usuarios comuns, com fotos de festas e afins.
  Uma página para republicanos, com toda uma trama de agendamento de visitas e afins.
  Uma página para os usuarios que estão na república em sí, para centralização da estrutura da república.

<!-- Apresentar restrições de funcionalidades e de escopo. -->
### 3. Restrições

  Neste trabalho não serão consideradas as API's de sistemas externos, como o sistema de vendas de ingresso e possivelmente a do whatsapp.
